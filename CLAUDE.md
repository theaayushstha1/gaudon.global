### EmailJS Setup Required
After removing Base44, EmailJS needs to be configured:

1. Sign up at https://www.emailjs.com/
2. Create a service (Gmail recommended)
3. Create an email template with variables:
   - `{{company_name}}`, `{{contact_person}}`, `{{email}}`
   - `{{phone}}`, `{{country}}`, `{{product_interest}}`
   - `{{quantity}}`, `{{message}}`, `{{urgency}}`
4. Get your Service ID, Template ID, and Public Key
5. Replace placeholder values in these files:
   - `src/pages/Contact.jsx`
   - `src/components/products/TechnicalLibraryModal.jsx`
   - `src/components/shared/StickyQuoteWidget.jsx`
